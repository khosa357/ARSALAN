# Aurum Intel — Autonomous Macro Risk Aggregator Engine

Aurum Intel is an entirely automated, hands-free macroeconomic intelligence platform engineered to track, aggregate, and analyze global market factors impacting **Gold (XAUUSD)** and **Bitcoin (BTCUSD)**. 

The system operates autonomously in the background: monitoring asset pricing trends, pulling contextual fundamental news wire items, and orchestrating analytical workflows through **Claude (Anthropic)** to generate high-value, structured trade frameworks completely free of manual human maintenance.
